# MePrep 🍴

## Overview

MePrep is a responsive meal prep assistant built with HTML, CSS, and Vanilla JavaScript.

The application helps users:

- Track groceries at home
- Save digital recipe cards
- Manage shopping lists
- Compare recipe ingredients against pantry items
- Automatically add missing ingredients to the shopping list
- Use voice input for faster interaction

This project was designed to simplify meal prep for busy individuals while demonstrating frontend development skills using Vanilla JavaScript.

---

# Features

## Grocery Inventory

- Add grocery items
- Remove grocery items
- Voice-assisted grocery input
- Persistent storage using localStorage

## Digital Recipe Cards

- Create recipes
- Save ingredients and instructions
- Remove recipes
- Voice-assisted recipe creation

## Smart Recipe Assistant

- Compare recipe ingredients against groceries
- Detect missing ingredients
- Automatically add missing items to shopping list
- Recipe readiness feedback

## Shopping List

- Add shopping items
- Remove shopping items
- Mark items complete
- Auto-populate missing recipe ingredients

## Responsive Design

- Mobile-friendly layout
- Hamburger navigation menu
- Smooth scrolling navigation
- Fade-in card animations

## Voice Recognition

- Web Speech API integration
- Microphone listening animation
- Voice input for:
  - Groceries
  - Recipes
  - Shopping list



```text

```
